## 🚀 Project Overview
This project performs an industrial-grade audit of airplane crash data to uncover deep-seated risks. Moving beyond simple fatality counts, this analysis focuses on **structural survivability** and **causal text mining** to determine why accidents happen and how likely passengers are to survive them.

### 🎯 Key Strategic Insights
* **The Safety Paradox:** 2019 data confirms that while crashes are rarer, they are becoming "Mass Fatality" events due to increased aircraft capacity.
* **NLP Causal Mining:** Automated extraction of accident triggers from historical summaries.
* **Third-Party Risk:** High-fidelity analysis of "Ground Fatalities" in urban aviation zones.
